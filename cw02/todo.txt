1. Зарегистрироваться на гитхабе
2. Сделать форк репозитория ...
3. Найти строку для клонирования на странице созданного репозитория
4. git clone эта-строка (вставка Shift+Ins, или Ctrl+Shift+V или через контекстное меню)
5. ls; cd название-репозитория
6. посмотрите на историю: git log; gitk
7. поредактируйте файл  introduction.doc (soffice introduction.ods)
8. сохраните и закоммитьте git commit -a -m '...'
9. отправьте изменения обратно: git push
10. найдите на сайте кнопку create pull request и нажмите её
----
Для скучающих
11. Создайте пустой репозиторий (mkdir repo; cd repo; git init)
12. Создайте в нём файл и несколько коммитов
13. Переключитесь на другую ветку (git branch name; git checkout name)
14. Создайте несколько коммитов
15. Переключитесь на ветку master (git checkout master)
16. Создайте несколько коммитов
17. Посмотрите в gitk на ваш репозиторий
18. Выполните git merge; git commit -a -m '...'
19. Посмотрите в gitk на ваш репозиторий сейчас
