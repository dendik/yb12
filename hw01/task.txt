Адрес дистрибутива git: http://git-scm.com/download

Если его установить под Windows (и оставить все настройки по умолчанию), то в контекстном меню появится пункт git bash. В Linux git устанавливается через стандартную систему пакетов (например, sudo apt-get install git в производных Debian). Инсталлятор с сайта git под MacOS делает команду git доступной в командной строке консоли.

С этого этапа вся работа с git на любой системе эквивалентна тому, как с ним работают в Linux, то есть то, как мы это делали в классе.

Часть 1.
1. Регистрируемся на github.com У кого уже есть аккаунт -- этот шаг можно пропустить
2. Настраиваем ssh-ключ и скармливаем его гитхабу (кто им уже пользуется -- опять же, можно пропустить этот шаг)
3. Делаем fork проекта: https://github.com/victor-yacovlev/hse-programming-2015
4. Делаем клон своей копии проекта на локальный компьютер
5. В проекте сейчас находится файл introduction.doc, который содержит таблицу. Предлагается ее заполнить, и закоммитить.
6. Делаем git push с локального компьютера на гитхаб
7. Создаём pull-request через веб-интерфейс гитхаба.
